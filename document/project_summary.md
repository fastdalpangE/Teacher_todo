# Teacher Daily Project Summary

이 문서는 **Teacher Daily** 프로젝트의 구현 내역 및 작업 로그를 정리한 문서입니다.

## 📅 작업 이력
| 날짜 | 주요 작업 |
|------|---------|
| 2026-04-11 | 프로젝트 초기화, React + Vite 구성, 기본 CRUD 시스템 구축 |
| 2026-04-11 | 수업 수정/삭제 기능, 전역 모달 상태 관리 구현 |
| 2026-04-28 | 시간표 전면 개편 (9~18시 + 학생별 필터) |
| 2026-04-28 | 할일·전달 페이지 완전 구현 (5개 섹션) |
| 2026-04-28 | 더미 데이터 확장 (18명 학생, 12수업, 3학교) |
| 2026-04-28 | Store 확장 (11개 신규 액션 + localStorage 머지) |

## ✅ 완료된 기능

### 1. 프로젝트 기반
- **Vite + React 18**: 핫 리로드 개발 환경
- **전역 상태 관리**: Context API + useReducer, 22개 액션 지원
- **자동 저장**: localStorage 기반 자동 저장, 신규 필드 자동 머지
- **Noto Sans KR**: 한국어 최적화 웹 폰트

### 2. 수업일지 탭 (SchedulePage)
- **주간/월간/일간 뷰 전환**
- **수업 필터**: 학교별, 학년별, 과목별
- **수업 카드 (ClassCard)**: 수업 정보 + 학생 태그 + 수정/삭제
- **학생별 기록 관리**: 출석(출석/지각/결석), 단어 오답, 강사 코멘트
- **전역 수업 모달**: 등록/수정 통합 처리
- **월간 달력**: 날짜별 수업 건수 표시, 클릭 시 일간 뷰 이동

### 3. 할일·전달 탭 (TodoPage) ⭐ NEW
- **5개 섹션 타입**: 할일, 전달사항, D-Day, 월말 보고서, 자유 메모
- **할일/전달사항**: 추가, 체크/해제, 삭제 (체크 애니메이션 포함)
- **D-Day 관리**: 커스텀 D-Day 추가 + 학교 시험 일정 자동 통합
  - D-n 자동 계산 배지 (색상 구분: 당일/7일 이내/이후/지남)
- **월말 보고서**: 전체 학생 칩 목록
- **자유 메모**: 추가/삭제/이름변경/내용 저장 가능
- 모든 섹션 접기/펼치기 토글

### 4. 학생·학교 탭 (ManagePage)

#### 시간표 (TimetableGrid) ⭐ UPDATED
- **9:00~18:00 전체 시간 범위** 고정 표시
- **색상 코딩**: 리딩(파랑), 문법(보라), 듣기(청록), 파닉스(초록), 쓰기(노랑)
- **수업 블록**: 이름 + 종류 + 학생 수 + 교재 표시
- **오늘 요일 하이라이트**: 빨간색 헤더
- **종류 범례**: 상단 범례 바
- **학생별 시간표**: 드롭다운 + 학생 칩으로 개별 시간표 조회

#### 학생 관리 (StudentList)
- 학생 프로필 카드 (아바타, 학년, 학교, 진도메모)
- CRUD (추가/수정/삭제)
- 히스토리 조회 (수업 기록 타임라인, 출석/단어/코멘트)

#### 학교 관리 (SchoolList)
- 학교 카드 + 시험 일정 목록
- CRUD + 시험 일정 관리 (학년/종류/날짜/범위)

### 5. CSV 데이터 임포트
- CSV 파일 업로드 + 안전 파싱
- 미리보기 테이블 + 오류 검증

### 6. 디자인 시스템
- Glassmorphism 기반 프리미엄 UI
- 반응형 레이아웃 (모바일/태블릿/데스크탑/와이드)
- 부드러운 전환 애니메이션
- 다크 네비게이션 + 카드 기반 레이아웃
